# Genesis - Second Brain System

A unified GTD + knowledge management system built for AI-assisted productivity.

## Features

- **Dashboard** - GTD stats at a glance
- **Actions** - Task management with contexts
- **Projects (Mission Control)** - Strategic project tracking
- **Memory** - Knowledge base / reference storage
- **CRM** - Contact management

## Stack

- **Frontend**: React + TypeScript + Vite + Tailwind
- **Backend**: Express.js + SQLite

## Quick Start

```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm run dev
```

## API

Backend runs on port 3001. API routes:
- `/api/v1/dashboard` - Dashboard stats
- `/api/v1/actions` - Task CRUD
- `/api/projects/mission-control` - Project tracking
- `/api/v1/memory` - Knowledge entries
